# Mesa — Photo Guide

Mesa is the **Dine** archetype in the Apotome website program.

## Where do photos go?

Drop final, web-optimized JPGs into `public/photos/` using the **exact filenames** below.

## Variants

| Variant      | Total photos | Gallery shots |
| ------------ | ------------ | ------------- |
| Essentials   | ~8           | 6             |
| Portfolio    | ~16          | 12            |

## Required (Essentials)

| Filename            | Shot                                            |
| ------------------- | ----------------------------------------------- |
| `hero.jpg`          | Signature exterior or hero dish, golden hour    |
| `about-room.jpg`    | Wide dining room, warm light                    |
| `dish-01.jpg`       | Top-down hero plate                             |
| `dish-02.jpg`       | Lighter dish, 45° angle                         |
| `dish-03.jpg`       | Centerpiece / large-format dish                 |
| `interior-01.jpg`   | Bar / counter detail                            |
| `interior-02.jpg`   | Window or doorway with daylight                 |
| `team-01.jpg`       | Chef or staff at work                           |

## Photo specs

- 2400 px on the long edge minimum
- Color-graded JPG, sRGB, quality ~80
- No watermarks or logos
- Horizontal where possible; the gallery handles tall shots automatically

## Tips

- Shoot during the **golden hour** (1 hour after sunrise / before sunset)
- Use available light; avoid on-camera flash
- Capture **hands and motion** — pouring, plating, chopping
- Keep brand **colors consistent** across the set
